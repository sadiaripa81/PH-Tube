1: Discuss the scope of var, let, and const

Variables are container for storing data.There are three type of variables.

Var

    Variables declared with 'var' are hoisted to global scope.
    Variables declared with 'var' is moved to the top of the function.
    'var' variables are redeclarable.
    Value can be reassigned.
    'var' can be used accessed it is declared but it will have value undefined.

Let

    Variables declared with 'let' are only accessible within the block in which they are defined.
    'let' variables are not redeclarable
    Value can be reassigned.
    let variables are not accessible outside of the block where they are declared.
    Can not use or accessed it before declaration.

Const

    'const' variables are block-scoped they are not accessible outside of the block where they are declared.
    'const' variables are not redeclarable.
    Value can not be reassigned.
    Can not use or accessed it before declaration.

2: Cases of null and undefined

Undefined and Null both are primitive values in JavaScript.But herees are some difference between them :

undefined

    It's a way to signal that a value is unintentionally empty or missing.
    The type is undefined.
    Equal to NUN (Not A Number ).
    Js return undefied when using a variable before it is declared.
    When a variable is declared but not assigned a value.

Null

    It's a way to signal that a value is intentionally empty or missing.
    No values on purpose.
    The type is object.
    Equal to 0 (zero).
